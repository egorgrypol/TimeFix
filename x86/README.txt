TimeFix

TimeFix is a Windows utility for system time correction, activity tracking,
and simple reminders.

--------------------------------------------------

FEATURES
--------

Activity
- Tracks how you use applications
- Shows usage time in minutes
- Displays usage percentage per application

TimeFix (Time Correction)
- Fixes and synchronizes system time
- Auto Mode: fixes time automatically
- Manual Mode: requires manual запуск settime
- Notify Mode: tray notifications about actions
- Silent Mode: works without notifications

Reminder
- Simple reminder system
- Custom reminder text
- Time-based notifications
- Tag support for organizing reminders

--------------------------------------------------

USAGE
-----
Launch TimeFix and use the interface to configure needed features.
Some functions may require administrator privileges.

--------------------------------------------------

SYSTEM COMPATIBILITY
--------------------
- Windows XP and newer
- 32-bit and 64-bit systems
- Designed to work on a wide range of Windows versions

--------------------------------------------------

INTERFACE & PERFORMANCE
-----------------------
- Clean and simple user interface
- Optimized for low system resource usage
- Stable performance even on older systems
